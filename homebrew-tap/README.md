# Homebrew Tap for Amanda OS CLI Tools

Homebrew tap oficial para las herramientas CLI de Amanda OS.

## Instalación

```bash
# Agregar el tap
brew tap corinoah1013/amanda-cli

# Instalar amanda-watch
brew install amanda-watch
```

## Actualizar

```bash
brew update
brew upgrade amanda-watch
```

## Desinstalar

```bash
brew uninstall amanda-watch
brew untap corinoah1013/amanda-cli
```

## Herramientas Disponibles

| Herramienta | Descripción |
|-------------|-------------|
| `amanda-watch` | Monitor de procesos y recursos |
| `amanda-logs` | Análisis de logs (próximamente) |
| `amanda-mail` | Cliente IMAP (próximamente) |
